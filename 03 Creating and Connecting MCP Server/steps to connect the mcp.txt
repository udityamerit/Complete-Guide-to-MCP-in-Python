Step 1: install the uv and initialized the project using the command of "uv init" the folder is initialized as project.

Step 2: install the mcp dependency using the "uv add mpc[cli]".

Step 3: create a file of your mcp server like "xyz.py" import the FastMCP server and initialized the mcp server and the its tools for this see the "weather.py".

Step 4: connect to your claude go to the setting and click on the developer settings and click on the edit file and open the claude_json and paste the given code :
{
  "mcpServers": {
    "weather": {
      "command":"uv",
      "args": ['
        "--directory",
        "weather.py",
        "run",
        "file path of your server file"
      ]
    }
  }
}

Note: an easy way to connect the locally created mcp server to the claude or any mcp host activate the env and the run "mcp install 'file.py'" it is automatically install the locally created server and edit the developer settings and you directly use the you locally build mcpServers.
