🔐  WHAT IS CRYPTOGRAPHY IN CRYPTO?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Cryptography is the science of securing information through
mathematical algorithms. In the context of cryptocurrencies,
it serves three core purposes:

  1. Security       — Protects transactions from tampering
  2. Anonymity      — Masks user identities via public/private keys
  3. Decentralisation — Eliminates the need for central authorities
                        (banks, governments)

Key Concepts:
  • Public Key  : Your wallet address (shareable)
  • Private Key : Your secret password (never share)
  • Hash Function: Converts data into a fixed-length code (e.g. SHA-256)
  • Blockchain   : A chain of cryptographically linked transaction blocks
